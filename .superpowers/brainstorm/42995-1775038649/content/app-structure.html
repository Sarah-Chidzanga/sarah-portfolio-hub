Request Flow — a page visit
Browser → API Gateway → Lambda
↓ Mangum converts event to WSGI
Flask route → db.py → DynamoDB
↓ Jinja2 renders template
HTML response → Browser
HTMX Interaction Flow — a like click
1. Visitor clicks ❤ button
HTMX sends POST /like/project/<slug>
2. Flask route increments DynamoDB
db.increment_likes("projects", slug)
3. Flask returns just the button HTML
renders partials/like_button.html
4. HTMX swaps the button in-place
❤ 12 → ❤ 13, heart bounces (CSS)
Key idea — partials: The templates/partials/ folder holds small HTML snippets that HTMX swaps in without reloading the page. The like button, comment list, and visit counter are all partials — Flask renders them and HTMX drops them into the right spot.
